Use of AccessibilityService API:


Zen Detox Phone leverages the AccessibilityService API to provide core features, such as app reminders and app blocking. With your explicit consent, this API enables the app to display a reminder screen when the allocated time for a time-sensitive app has lapsed. These features are designed to help you maintain focus and productivity.
